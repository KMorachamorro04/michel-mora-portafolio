## 🛠️ Tecnologías y Herramientas
Este portafolio fue construido utilizando un flujo de trabajo moderno:
* **Vite & Node.js:** Para un entorno de desarrollo rápido y eficiente.
* **JavaScript (ES6+):** Lógica para la navegación y componentes.
* **CSS3:** Diseño responsivo con efectos de vanguardia como Glassmorphism.
* **IA Integration:** Uso de herramientas de Inteligencia Artificial para optimizar el código y el diseño.
